# Content Management Guide

A guide for managing the markturnerjazz.org website. Written for use on Android phone or any web browser.

## Logging In

1. Open your browser and go to `markturnerjazz.org/config` (or `squarespace.com` and log in)
2. Enter your email and password
3. You'll see the site editor dashboard

Tip: Bookmark the login page on your phone's home screen for quick access.

## Adding a Tour Date

1. Log in to Squarespace
2. Tap **Pages** in the left menu
3. Tap **Tour Dates**
4. Tap the **+** button to add a new event
5. Fill in:
   - **Title**: Venue name and city (e.g., "Village Vanguard, New York")
   - **Date & Time**: When the performance is
   - **Location**: Full venue address
   - **Description**: Any details (other musicians, ticket info)
6. To add a ticket link: add a button block in the description with the ticket URL
7. Tap **Save** or **Publish**

To edit an existing date: tap on it from the Tour Dates page and make changes.

To remove a past date: past events archive automatically. No action needed.

## Updating Your Bio

1. Log in to Squarespace
2. Tap **Pages** → **About**
3. Tap on the text you want to edit
4. Make your changes directly in the text editor
5. Tap **Save**

## Adding Press Quotes

1. Navigate to **Pages** → **About**
2. Scroll to the press quotes section
3. Tap **Edit** on the section
4. Add a new text block with the quote
5. Format: *"Quote text here"* — Publication Name
6. Tap **Save**

## Managing the Store

### Adding a Product

1. Tap **Commerce** in the left menu (or **Pages** → **Store**)
2. Tap **+** to add a new product
3. Select **Physical Product**
4. Fill in:
   - **Name**: Product name (e.g., "Lathe of Heaven CD")
   - **Price**: Set the price
   - **Description**: Brief description
   - **Images**: Tap to upload product photos
   - **Inventory**: Set quantity available (optional)
5. Tap **Save & Publish**

### Updating Inventory or Price

1. Go to **Commerce** → **Products**
2. Tap on the product
3. Update the price or inventory count
4. Tap **Save**

### Viewing Orders

1. Tap **Commerce** → **Orders**
2. View order details, customer info, and shipping address
3. After shipping an item, mark the order as **Fulfilled**

## Uploading Photos

1. Navigate to the page where you want the photo
2. Tap **Edit** on the section
3. Tap **+** to add an image block
4. Select or drag in your photo
5. Add alt text (a brief description of the photo for accessibility)
6. Tap **Save**

Photo tips:
- Use landscape orientation for hero images
- Square photos work best for album artwork and product images
- Minimum 1500px wide for good quality on all screens

## Adding Music Links

1. Navigate to **Pages** → **Music**
2. Tap **Edit** on the album you want to update (or add a new section)
3. Add button blocks with links to:
   - Bandcamp page
   - ECM Records page
   - Spotify artist page
   - Apple Music artist page
4. Tap **Save**

## Quick Tips

- **Preview changes**: Before publishing, use the preview button to see how it looks
- **Mobile editing**: The Squarespace app is available but the mobile browser works too
- **Undo mistakes**: Squarespace keeps version history — you can revert changes
- **Need help?**: Squarespace has 24/7 support via chat at support.squarespace.com
